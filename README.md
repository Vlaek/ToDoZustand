# Installation

1. **Cloning a repository**:

   ```bash
   git clone https://github.com/Vlaek/ToDoZustand.git
   cd ToDoZustand
   ```

2. **Installing dependencies**:

   ```bash
   npm install
   ```

3. **Launching the application**:
   ```bash
   npm start
   ```
